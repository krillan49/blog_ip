# simpleblog(itproger)


### учебный проект с [itproger](https://itproger.com/course/ruby-on-rails)


* Ruby version 3.1.2p20
* Rails version 7.0.7


## Start project

1. Install gems:
```bash
bundle install
```

2. Migrate:
```bash
rake db:migrate
```

3. Run RoR application:
```bash
rails s
```

4. Open in your browser: 
```bash
http://localhost:3000/
```
